# Multi-Agent Debugger Workflow

## Overview
This project implements an LLM-based multi-agent system designed to automatically detect, analyze, and fix code errors. The system uses multiple agents that collaborate and refine outputs through feedback loops to improve debugging accuracy.

## Key Features
- Automated code error detection and correction  
- Multi-agent architecture for collaborative problem solving  
- Feedback loops to iteratively improve debugging results  
- Scalable and modular workflow design  

## Tech Stack
- Python  
- Large Language Models (LLMs)  
- Agentic AI  

## How It Works
1. Input code is provided to the system  
2. Multiple agents analyze and identify errors  
3. Suggested fixes are generated  
4. Feedback loops refine the output for better accuracy  

## Results
The system improves debugging efficiency by providing structured and iterative solutions to coding errors.

## How to Run
- Open the notebook in Google Colab  
- Run all cells step-by-step  
- Provide sample code input to test debugging
